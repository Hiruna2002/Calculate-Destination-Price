A modern, responsive, and user-friendly delivery price calculator built with React, TypeScript, and Tailwind CSS. This project allows users to enter a distance in kilometers and instantly calculate the delivery price. The interface is mobile-first, sleek, and intuitive for all devices.

🔹 Features
    Real-time price calculation based on distance
    Mobile-first responsive design with Tailwind CSS
    Modern UI with gradient headers, rounded cards, and smooth buttons
    Clear button to reset inputs instantly
    Type-safe with TypeScript for better developer experience
    Easy to extend for additional pricing rules
